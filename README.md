# Currency Converter

## Автор: Артур Афанасьев

## Описание

Простое GUI-приложение для конвертации валют с использованием внешнего API и сохранением истории операций.

## Требования

- Python 3.x
- Библиотеки: `requests`, `tkinter`
- API-ключ с [exchangerate-api.com](https://www.exchangerate-api.com/)

## Как получить API-ключ

1. Зарегистрируйтесь на сайте [exchangerate-api.com](https://www.exchangerate-api.com/).
2. Получите бесплатный или платный API-ключ.
3. Вставьте ключ в переменную `API_KEY` в файле `currency_converter.py`.

## Использование

1. Запустите скрипт:
   ```bash
   python currency_converter.py
   ```
2. Выберите валюты, введите сумму и нажмите «Конвертировать».
3. История операций сохраняется в файл `history.json`.

## Примеры использования

- Конвертация 100 USD в EUR.
- Проверка истории последних 5 операций.
- Очистка истории (удалите файл `history.json`).
